# BlindCheck Quick Start Guide

## What is BlindCheck?

BlindCheck is an iOS app for validating cemetery marker placements. It uses your iPhone camera to scan markers, read names using OCR, and confirm they match the cemetery's records.

## Field Guide

### Before You Start
- âœ“ Good lighting (avoid direct sun glare)
- âœ“ Clean markers if possible
- âœ“ Charged device

### Scanning Tips
1. Stand 3-4 feet from marker
2. Hold device steady
3. Align marker in center of grid
4. Capture when text is clear and readable
5. Wait for green checkmark

### Best Practices
- Work systematically (row by row)
- Add notes for any discrepancies
- Double-check uncertain validations
- Export report at end of session

## Navigation

### Moving Between Markers
- Tap any marker in grid to center on it
- Use Status tab to jump to specific markers
- Grid automatically shows surrounding markers

### Finding Information
- **Marker Name**: Top of grid cell
- **Position**: (Row, Column) coordinates
- **Status**: Colored circle indicator

## Common Workflows

### Daily Validation Session
1. Open app at cemetery section
2. Go to Grid tab, select starting position
3. Switch to Scan tab
4. Scan center marker and surrounding markers
5. Return to Grid tab, validate results
6. Move to next position
7. Repeat until section complete
8. Export report

### Quick Spot Check
1. Status tab â†’ Find marker
2. Tap to open in Grid
3. View details
4. Manual validation if needed

### Fixing Errors
1. Find marker in Status or Grid tab
2. Review validation notes
3. Re-scan if needed
4. Update validation with correct status
